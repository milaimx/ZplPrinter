使用说明

ZPL Studio v2 专为处理双栏标签文件设计，参考 Labelary.com 渲染引擎。

🔑 核心功能：双栏解构

你的文件每个 ^XA...^XZ 块包含左右两列标签：

  • 左列：x坐标 < 400
  • 右列：x坐标 ≥ 430，自动平移到左侧

点击 ✂️ 解构双栏 后，自动拆出所有单标签并批量渲染预览。

📄 PDF 导出

每个标签独占一页，精确匹配标签尺寸。支持一键下载全部。

🌐 渲染模式

联网时调用 Labelary API 获取像素级精确预览；断网自动切换离线 Canvas 渲染。
